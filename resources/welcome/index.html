Create a New Truffle Tools Service

To create a new Truffle Tools Service if you don't have one.
  1. From the Command Palette, or Truffle Tools tab, click on "Create a new network" and select Truffle Tools Service
If you have an existing Truffle Tools Service, you may connect to it.
  1. From the Command Palette, or Truffle Tools tab, click on "Connect to network" and select Truffle Tools Service

Create a Smart Contract and Project Folder

To create a basic smart contract folder structure and contract.
  1. From the Command Palette click on "New Solidity Project"

Interact with your Smart Contract

To interact with your contract
  1. Deploy your contract locally
  2. Right click on your contract .sol and select "Show Smart Contract Interaction Page"
